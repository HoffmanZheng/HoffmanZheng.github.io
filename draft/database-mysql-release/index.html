Database：[译] MySQL 5.6 5.7 8.0 Release Notes

Written by Chenghao Zheng with ♥ on December 19, 2022 in Study notes ,99 words

MySQL 在 1996 年发布了第一个官网正式发行版 3.11.1，虽在 2008 年和 2009 年先后被 Sun 和 Oracle 收购，但凭借着其性能好、成本低、可靠性好，其在网站开发中被广泛使用。根据 DB-Engines 对各个数据库产品的综合排名，MySQL 与 Oracle、SQL Server 自 2013 年起就稳定地占据着 最受欢迎的数据库管理系统 的前三名。

相关阅读：MySQL 的前世今生

自 2009 年 Oracle 以 74 亿美元收购 Sun Microsystems 后，MySQL 随后又在 2010-12-03 发布了 5.5 GA 版本，2013-02-05 发布了 5.6 GA 版本，2015-10-21 发布了 5.7 GA 版本，2018-04-19 发布了 8.0 GA 版本。基于市面上 MySQL 的中文学习书籍比如 高性能 MySQL 和 MySQL技术内幕 对新版本的 MySQL 的新特性描述较少，笔者本着学习的态度翻译 5.6 5.7 8.0 版本的 MySQL Release Notes，如有不足之处，欢迎批评指正。
